# Frontend Stack

The frontend uses:

- Vite 5
- React 18
- TypeScript 5
- Tailwind CSS
- Zustand
- localforage
- lucide-react
- Vitest
- Playwright

The top-level application shell is `src/App.tsx`. It is currently large and should be refactored cautiously, with behavior-preserving extractions preferred over broad rewrites.
