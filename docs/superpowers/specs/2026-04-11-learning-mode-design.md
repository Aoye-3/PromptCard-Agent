# 学习模式静态版本设计文档
## 文档信息
- **版本**：1.0
- **更新时间**：2026-04-11
- **参考依据**：Google Veo 3.1官方提示词指南
- **内容结构**：9个PromptCard对应独立教学案例，完全匹配卡片类型

---
## 核心设计思路
学习模式采用"一卡一例"结构，每个卡片类型对应一个独立的教学模块，包含：
1. 正反案例对比
2. 核心学习要点
3. 四步优化过程演示
所有内容均来自官方最佳实践，确保专业性和可操作性。

---
## 教学内容详情
### 1. Subject（主体）卡片
**教学核心**：如何描述清晰、具体的主体特征，避免模糊、笼统的指代
| 对比维度 | 内容 |
|----------|------|
| 反面示例 | 一个人 |
| 正面示例 | 穿运动服的年轻女性，背着皮质挎包，棕色凌乱头发扎成马尾，脸上有雀斑 |
| 学习要点 | • 明确身份属性：年龄、职业、穿着、外貌特征<br>• 添加个性化细节：专属道具、标志性特征<br>• 避免模糊指代：不用"一个人"、"某物体"这类泛称 |
| 优化过程 | 1. 基础：一个人<br>2. 进阶：年轻女性<br>3. 完善：穿运动服的年轻女性，背着皮质挎包<br>4. 最佳：穿运动服的年轻女性，背着皮质挎包，棕色凌乱头发扎成马尾，脸上有雀斑 |

### 2. Action（动作）卡片
**教学核心**：如何描述生动、具体的动作行为，包含情绪和细节
| 对比维度 | 内容 |
|----------|------|

| 优化过程 | 1. 基础：在走路<br>2. 进阶：推开丛林藤蔓走路<br>3. 完善：推开巨大的丛林藤蔓，露出隐藏的路径<br>4. 最佳：推开巨大的丛林藤蔓，露出隐藏的路径，表情充满敬畏，手抚过断壁残垣上复杂的雕刻 |

| 优化过程 | 1. 基础：在山里<br>2. 进阶：在峡谷里<br>3. 完善：巨大的雾气弥漫的峡谷边缘，正值日出时分<br>4. 最佳：巨大的雾气弥漫的峡谷边缘，正值日出时分，古老的苔藓覆盖的遗迹，周围是茂密的丛林 |

### 8. Timing（时序）卡片
**教学核心**：如何使用时间戳分段控制多镜头视频节奏
| 对比维度 | 内容 |
|----------|------|
| 反面示例 | 拍一个探险视频 |
| 正面示例 | [00:00-00:02] 后背中景，探险家推开藤蔓<br>[00:02-00:04] 反打镜头，探险家表情震惊看着遗迹<br>[00:04-00:06] 跟拍镜头，探险家走入空地抚摸雕刻<br>[00:06-00:08] 广角升降镜头，展现整个神庙建筑群 |
| 学习要点 | • 时间格式：`[HH:MM:SS-HH:MM:SS]` 分段标记<br>• 镜头切换：每个时间段对应一个独立镜头<br>• 节奏控制：根据内容调整每个镜头的时长 |
| 优化过程 | 1. 基础：拍一个探险视频<br>2. 进阶：拍8秒的探险视频，分4个镜头<br>3. 完善：[0-2秒]拍后背，[2-4秒]拍脸部，[4-6秒]拍走路，[6-8秒]拍全景<br>4. 最佳：完整时间戳分段示例（如上） |

### 9. Constraint（约束）卡片
**教学核心**：如何使用负面提示词排除不需要的内容，精准控制生成结果
| 对比维度 | 内容 |
|----------|------|
| 反面示例 | 不要奇怪的东西 |
| 正面示例 | 荒凉的景观，没有建筑物或道路，禁止出现AI畸形手，人物一致性优化开启，16:9宽屏，分辨率1080P，高细节度，低变形 |
| 学习要点 | • 正向描述排除内容：用"没有建筑物或道路"代替"不要人造结构"<br>• 技术参数约束：分辨率、比例、质量要求<br>• 质量控制：禁用畸形、低质、变形等常见问题 |
| 优化过程 | 1. 基础：不要奇怪的东西<br>2. 进阶：不要人造建筑，保持人物正常<br>3. 完善：荒凉的景观，没有建筑物或道路，禁止出现AI畸形手<br>4. 最佳：荒凉的景观，没有建筑物或道路，禁止出现AI畸形手，人物一致性优化开启，16:9宽屏，分辨率1080P，高细节度，低变形 |

## 交互设计规范
### 页面结构
1. 顶部下拉选择器：选择要学习的卡片类型（9种可选）
2. 案例对比学习区域：左右分栏展示反面/正面示例
3. 学习要点区域：蓝色背景卡片展示核心知识点
4. 展开过程演示区域：4个步骤卡片展示从差到优的优化过程

### 交互逻辑
- 默认展示第一个（Subject）卡片的教学内容
- 用户通过顶部下拉切换不同卡片的教学内容
- 所有内容静态展示，无需后端接口支持
